1.**Shell 基础**

Shell 是用户与 Linux 内核的交互接口，默认使用 bash。

脚本以 `#!/bin/bash` 开头，后缀 `.sh`。

执行方式：`chmod +x xxx.sh` 后 `./xxx.sh` 或直接 `bash xxx.sh`。

 2.**Shell 变量**

定义：`name="value"`（等号两边无空格）。

使用：`$name` 或 `${name}`。

环境变量：`$HOME`、`$USER`、`$PATH` 。

只读变量 `readonly`、删除变量 `unset`。

3.**参数传递**

$0` 脚本名，`$1 $2` 位置参数，`$#` 参数个数。

$@` 所有参数，`$?` 上一条命令返回值。

4.**数组操作**

定义：`arr=(1 2 3 4 5)`

读取：`${arr[0]}`、`${arr[@]}`

长度：`${#arr[@]}`

5.**运算符**

算术运算：`$((a+b))`、`expr $a + $b`

关系判断：`-eq -ne -gt -lt`

字符串判断：`=` `!=` `-z` `-n`

文件测试：`-f` 文件、`-d` 目录、`-r` 可读

6.**输出命令**

echo` 简单输出

printf` 格式化输出

7.**流程控制**

if/elif/else` 条件判断

for` 循环遍历

while` 条件循环

case` 多分支选择

break`/`continue` 控制循环

8.**函数**

定义：`func() { ... }`

调用：直接写函数名

参数：`$1 $2` 同脚本参数

9.**重定向**

` 覆盖写入，`>>` 追加

<` 输入重定向

&>` 标准输出 + 错误一起重定向

/dev/null` 丢弃输出

10.**文件包含**

使用 `. 文件名` 或 `source 文件名` 引入其他脚本，实现代码复用。
